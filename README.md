# Project 1: SQL Server to Python Automation 

# Overview
This project automates the extraction of luxury bike data from an AdventureWorks SQL Server database,
calculates UK VAT (20%), and generates both a CSV report and a visual trend chart.

# Tech Stack
**Database:** SQL Server (AdventureWorks2019)
**Language:** Python 3.13
**Libraries:** Pandas, PyODBC, Matplotlib

# Key Features
* Automated connection to local SQLEXPRESS instance.
* Dynamic VAT calculation column.
* Automated export to Excel-ready CSV and PNG Visualisation.
